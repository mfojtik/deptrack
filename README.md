## deptrack

This tool analyze Golang dependencies between repositories in multi-repo project and report the current state
of the dependencies.

### Installation

```console
go install github.com/mfojtik/deptrack
```

### Usage

The first parameter is path to the YAML configuration. Second parameter is root directory for Git repositories.
Last parameter is a path where to write the reports.
